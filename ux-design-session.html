  • 73-question assessment is above fatigue threshold (20 min max)
  • 83% completion for 1-3 questions vs 42% for 15+
  • Consider: tiered assessment or progressive profile building

Personalization Impact:

Design Weekly

Key Decisions

Decision 1: Home Page

Options: Keep home | Eliminate home | Merge with chat

Eliminate (Chat IS home)

Evidence: Slack CPO philosophy - "What is home for?"

Decision 2: Silhouette Gate

Options: Hard gate | Soft gate (3 messages) | No gate

Soft gate (3 messages, then require Silhouette)

Evidence: Let users experience Ali's value first, then prompt for deeper personalization.

Decision 3: Post-3-Message Prompt

Options: Modal | Inline banner | Toast notification

Inline banner (non-blocking but prominent)

Evidence: Shows value prop after user has experienced Ali, not before.

Decision 4: Placeholder Copy

Options: Generic | Action-oriented

"What negotiation are you working through?"

Evidence: NN/g research - specific prompts outperform generic ones.

Decision 5: Trait Typography

Options: Bold only | Bold + color | Underline

Bold only (accessible)

Evidence: Accessibility standards, visual hierarchy best practices.
